Privacy — Our Core Commitment

Core Privacy Commitments

• No phone numbers exposed

• No ads, no data resale

• No message history retained after expiry

• No third-party trackers

Message Lifecycle

• Message exists only during its timer window

• After expiration, message is permanently destroyed

• Sender cannot recover expired messages

• Receiver cannot save or export messages

User Control

• User chooses view duration

• Deletion is mandatory (cannot be disabled)

• Logout preserves data

• Delete account permanently erases identity

Compliance Language

• Built with privacy-by-default principles

• Data minimization approach

• Region-aware compliance readiness (Africa-focused)
