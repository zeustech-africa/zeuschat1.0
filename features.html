Features — Built for Secure Communication

Messaging & Security

• Zeus-PIN based identity — no phone number sharing

• PIN-to-view encrypted messages

• Mandatory self-destruct timers (5–45 seconds)

• Messages, files, and voice notes auto-delete

• Screenshot and screen-recording prevention

• No cloud message storage after expiry

Communication Tools

• Text messaging

• Voice notes (VN)

• File sharing (images, videos, documents)

• Emoji reactions and emoji picker

• Read receipts and delivery status

Privacy-by-Design

• Contacts added only by Zeus-PIN

• Mutual approval required before chat appears

• No discoverability without consent

• Offline / online presence control

Account Control

• Logout without data loss

• Account deletion permanently removes all data

• New registration generates a new Zeus-PIN

Platform Vision

• Built for Africa

• Designed for high-trust communication

• Lightweight, fast, and secure
